CRM系统结束分享
一个CRM系统考察的是一个对于python一级django的所有技术的综合能力
在一个以技术驱动的公司中这个东西才非常重要，需要考虑到项目中的代码复用性，功能复用性

稳扎稳打的走法
    初级阶段：增删改查 + linux
    阶段升级：做项目增加熟练度 + 开发组件 + linux
    阶段升级：看源码（flask+django） + linux
    终极阶段：
        架构（linux硬件+代码） + linux
        linux = 硬件 + linux
        代码 = 设计模式 + 领域驱动模型（代码可扩展） + linux
